Notes

We will be using PyQt to create the GUI for this desktop application. The desktop application should carry out the following processes:
a.) Automatically create an updated rewards/objectives spreadsheet ready for mail merging.
1.) User downloads the sales objectives update from Rick's email, exports this PDF to an Excel workbook (saving to the correct directory - *this directory to be determined later)
2.) User clicks the "Upload file to prep for mail merge" button in the program and selects this file
3.) The program will then process this workbook, grabbing the relevant data for each OSC and inputting that into a new file with the necessary contact information as well
b.) Automatically update emails in this mail merge spreadsheet
